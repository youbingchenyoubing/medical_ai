## 1. 医学影像发展历史

### 1.1 里程碑事件

| 年份 | 事件 | 意义 |
|------|------|------|
| 1895 | 伦琴发现X射线 | 首次无创观察人体内部,获1901年首届诺贝尔物理学奖 |
| 1917 | 拉东提出断层重建数学理论 | 为CT重建奠定数学基础 |
| 1971 | 豪斯菲尔德发明第一台CT扫描仪 | 首次实现断层成像,获1979年诺贝尔生理学或医学奖 |
| 1973 | 劳特伯和曼斯菲尔德提出MRI原理 | 无辐射成像技术,获2003年诺贝尔生理学或医学奖 |
| 1990s | PET/CT融合成像技术成熟 | 实现功能与解剖信息融合 |
| 2000s | 功能性MRI(fMRI)广泛应用 | 脑科学研究的重要工具 |
| 2010s | 人工智能革新医学影像分析 | 深度学习推动智能诊断 |

### 1.2 发展阶段

#### 第一阶段：投影成像时代（1895-1970s）
- **技术特点**：X射线投影成像
- **局限性**：
  - 三维结构压缩成二维图像
  - 组织重叠影响诊断
  - 无法区分密度相近的组织

#### 第二阶段：断层成像革命（1970s-1990s）
- **CT技术**：
  - 从多个角度采集X射线投影
  - 计算机重建三维结构
  - 首次清晰显示大脑内部结构
  
- **MRI技术**：
  - 利用氢原子核在磁场中的共振
  - 无电离辐射
  - 优越的软组织对比度

#### 第三阶段：功能与分子成像时代（1990s至今）
- **PET（正电子发射断层扫描）**：
  - 显示代谢活性
  - 肿瘤早期发现
  - 使用FDG示踪剂

- **功能性MRI（fMRI）**：
  - 检测血氧水平变化
  - 脑功能研究
  - 神经科学研究

## 2. 医学成像模态

### 2.1 X射线成像

#### 基本原理
- X射线穿透人体
- 不同组织对X射线吸收程度不同
- 形成投影图像

#### 临床应用
- 骨骼系统检查
- 胸部X光片
- 牙科X光
- 乳腺钼靶检查

#### 优缺点
| 优点 | 缺点 |
|------|------|
| 成像速度快 | 辐射暴露 |
| 成本低 | 软组织对比度差 |
| 设备普及 | 组织重叠 |

### 2.2 CT（计算机断层扫描）

#### 基本原理
```
X射线源 → 旋转扫描 → 探测器接收 → 数据采集 → 计算机重建 → 断层图像
```

#### 关键概念

**Hounsfield单位（HU）**
- 定义：CT值的标准化单位
- 水的CT值：0 HU
- 空气的CT值：-1000 HU
- 致密骨：+1000 HU

| 组织 | CT值范围（HU） |
|------|----------------|
| 空气 | -1000 |
| 脂肪 | -100 ~ -50 |
| 水 | 0 |
| 软组织 | +20 ~ +80 |
| 骨骼 | +300 ~ +3000 |

#### CT伪影

1. **射束硬化伪影（Beam Hardening Artifact）**
   - 原因：X射线穿过人体后能量谱改变
   - 表现：杯状伪影、条纹伪影
   - 解决：预过滤、软件校正

2. **金属伪影（Metal Artifact）**
   - 原因：金属植入物导致射线衰减
   - 表现：放射状条纹
   - 解决：迭代重建算法、金属伪影减少算法

3. **运动伪影（Motion Artifact）**
   - 原因：患者移动或呼吸
   - 表现：图像模糊、条纹
   - 解决：屏气扫描、快速扫描技术

#### 临床应用
- 头部CT：脑出血、脑梗死、颅骨骨折
- 胸部CT：肺结节、肺炎、肺癌筛查
- 腹部CT：肝脏病变、胰腺疾病
- CT血管造影（CTA）：血管病变诊断

### 2.3 MRI（磁共振成像）

#### 基本原理

```
强磁场 → 氢原子核排列 → 射频脉冲激发 → 共振信号 → 空间编码 → 图像重建
```

#### 关键概念

**K空间（K-space）**
- 定义：MRI原始数据的频率空间
- K空间中心：图像对比度信息
- K空间边缘：图像细节信息
- 傅里叶变换：K空间 → 图像空间

**脉冲序列（Pulse Sequence）**
- T1加权（T1-weighted）：解剖结构
- T2加权（T2-weighted）：病理改变
- FLAIR：抑制脑脊液信号
- DWI（弥散加权成像）：急性脑梗死

#### MRI优势
- 无电离辐射
- 优秀的软组织对比度
- 多平面成像
- 功能成像能力

#### 临床应用
- 神经系统：脑肿瘤、多发性硬化、脑梗死
- 肌骨系统：韧带损伤、软骨病变
- 腹部：肝脏、胰腺病变
- fMRI：脑功能定位

### 2.4 PET（正电子发射断层扫描）

#### 基本原理
```
注射FDG → 肿瘤细胞摄取增加 → 正电子发射 → 湮灭辐射 → 探测器接收 → 图像重建
```

#### FDG代谢
- FDG（氟代脱氧葡萄糖）：葡萄糖类似物
- 高代谢组织：肿瘤、炎症
- 低代谢组织：坏死组织

#### 临床应用
- 肿瘤诊断与分期
- 疗效评估
- 复发检测
- 神经系统疾病

### 2.5 PET-CT融合成像

#### 优势
- PET：功能代谢信息
- CT：解剖结构信息
- 融合：精准定位病灶

#### 临床价值
- 肿瘤诊断准确性提高
- 治疗方案制定
- 放疗靶区勾画

## 3. 医学影像数据格式

### 3.1 DICOM（Digital Imaging and Communications in Medicine）

#### 基本信息
- 标准：医学影像和相关信息的国际标准（ISO 12052）
- 用途：医学影像存储、传输、打印
- 包含：文件格式定义 + 网络通信协议

#### DICOM文件结构

```
DICOM文件
├── 文件头（File Meta Information）
│   ├── 文件前言（128字节）
│   ├── DICOM前缀（4字节）
│   └── 组长度
└── 数据集（Data Set）
    ├── Patient模块
    │   ├── Patient Name (0010,0010)
    │   ├── Patient ID (0010,0020)
    │   └── Patient Birth Date (0010,0030)
    ├── Study模块
    │   ├── Study Instance UID (0020,000D)
    │   ├── Study Date (0008,0020)
    │   └── Study Description (0008,1030)
    ├── Series模块
    │   ├── Series Instance UID (0020,000E)
    │   ├── Series Number (0020,0011)
    │   └── Series Description (0008,103E)
    └── Image模块
        ├── Rows (0028,0010)
        ├── Columns (0028,0011)
        ├── Pixel Data (7FE0,0010)
        └── Window Center/Width (0028,1050/1051)
```

#### DICOM Tag结构
- 格式：(组号,元素号)
- 示例：(0010,0010) = Patient Name
- VR（Value Representation）：数据类型
- VM（Value Multiplicity）：值的多重性

### 3.2 NIfTI（Neuroimaging Informatics Technology Initiative）

#### 基本信息
- 来源：NIH影像学信息工具倡议
- 主要应用：神经影像学、脑成像
- 扩展名：.nii 或 .nii.gz（压缩格式）

#### 与DICOM的区别

| 特性 | DICOM | NIfTI |
|------|-------|-------|
| 文件数量 | 每个切片一个文件 | 整个3D/4D体积一个文件 |
| 元数据 | 丰富的临床信息 | 简化的空间信息 |
| 用途 | 临床存储传输 | 科研分析处理 |
| 文件大小 | 较大 | 较小（压缩） |
| 处理效率 | 低（多文件） | 高（单文件） |

#### NIfTI文件结构
```
NIfTI文件
├── 头部（348字节）
│   ├── 数据维度（dim）
│   ├── 体素大小（pixdim）
│   ├── 数据类型（datatype）
│   ├── 空间变换矩阵（sform/qform）
│   └── 描述信息
└── 图像数据
    └── 二进制数据
```

### 3.3 其他格式

#### NRRD（Nearly Raw Raster Data）
- 特点：灵活的头部分离
- 用途：科学可视化

#### ANALYZE
- 组成：.img（数据）+ .hdr（头部）
- 局限：头部信息有限

#### MINC（Medical Imaging NetCDF）
- 基于NetCDF
- 支持多维数据

## 4. 图像重建算法

### 4.1 CT重建算法

#### 滤波反投影（Filtered Back Projection, FBP）

**原理**
```
投影数据 → 滤波 → 反投影 → 重建图像
```

**数学基础**
- Radon变换：投影 → 图像
- 逆Radon变换：图像 → 投影
- 傅里叶切片定理

**步骤**
1. 采集多个角度的投影数据
2. 对每个投影进行滤波（如Ram-Lak滤波器）
3. 将滤波后的投影反投影到图像空间
4. 累加所有反投影结果

**优缺点**
| 优点 | 缺点 |
|------|------|
| 计算速度快 | 对噪声敏感 |
| 算法成熟 | 伪影较多 |
| 临床应用广泛 | 低剂量成像质量差 |

#### 迭代重建算法（Iterative Reconstruction）

**原理**
```
初始图像 → 正向投影 → 比较误差 → 更新图像 → 迭代优化
```

**常用算法**
- ART（Algebraic Reconstruction Technique）
- SIRT（Simultaneous Iterative Reconstruction Technique）
- ML-EM（Maximum Likelihood Expectation Maximization）

**优势**
- 低剂量成像质量好
- 伪影减少
- 可建模物理过程

**缺点**
- 计算时间长
- 需要更多计算资源

### 4.2 MRI重建算法

#### 傅里叶变换重建

**原理**
```
K空间数据 → 2D/3D傅里叶变换 → 图像空间
```

**步骤**
1. K空间数据采集
2. 填充K空间
3. 傅里叶变换
4. 得到图像

#### 压缩感知重建（Compressed Sensing）

**原理**
- 利用图像稀疏性
- 从欠采样数据重建
- 加速成像

**条件**
1. 稀疏性：图像在某个变换域稀疏
2. 随机采样：K空间随机欠采样
3. 非线性重建：优化算法

**应用**
- 快速MRI
- 心脏MRI
- 动态成像

### 4.3 深度学习重建

#### 基于深度学习的CT重建
- 学习从低剂量到正常剂量的映射
- 去除伪影和噪声
- 提高图像质量

#### 基于深度学习的MRI重建
- 加速K空间采样
- 从欠采样数据重建
- 提高成像速度

## 5. 医学影像预处理

### 5.1 数据预处理

#### 窗宽窗位调整（Windowing）

**概念**
- 窗宽（Window Width, WW）：显示的CT值范围
- 窗位（Window Level, WL）：显示的中心CT值

**常用窗口**
| 窗口类型 | 窗位（HU） | 窗宽（HU） | 应用 |
|----------|-----------|-----------|------|
| 肺窗 | -600 | 1500 | 肺部病变 |
| 纵隔窗 | 50 | 350 | 纵隔结构 |
| 骨窗 | 300 | 1500 | 骨骼病变 |
| 脑窗 | 35 | 80 | 脑组织 |

**实现**
```python
def windowing(image, window_level, window_width):
    min_val = window_level - window_width / 2
    max_val = window_level + window_width / 2
    windowed = np.clip(image, min_val, max_val)
    normalized = (windowed - min_val) / (max_val - min_val)
    return normalized
```

#### 归一化（Normalization）

**方法**
1. **Min-Max归一化**
   ```python
   normalized = (image - image.min()) / (image.max() - image.min())
   ```

2. **Z-score归一化**
   ```python
   normalized = (image - image.mean()) / image.std()
   ```

3. **百分位归一化**
   ```python
   p1, p99 = np.percentile(image, [1, 99])
   normalized = np.clip(image, p1, p99)
   normalized = (normalized - p1) / (p99 - p1)
   ```

#### 重采样（Resampling）

**目的**
- 统一体素大小
- 降低计算量
- 多模态配准

**方法**
```python
import nibabel as nib
from scipy.ndimage import zoom

def resample_image(image, original_spacing, target_spacing):
    zoom_factor = np.array(original_spacing) / np.array(target_spacing)
    resampled = zoom(image, zoom_factor, order=1)
    return resampled
```

## 6. 深度学习在医学影像中的应用

### 6.1 医学图像分割（Segmentation）

#### U-Net系列

**U-Net（2015）**
- 编码器-解码器结构
- 跳跃连接
- 小数据集表现优秀

**架构**
```
编码器                    解码器
Conv → Pool    ─────→    Upsample → Conv
Conv → Pool    ─────→    Upsample → Conv
Conv → Pool    ─────→    Upsample → Conv
```

**变体**
- 3D U-Net：3D医学图像分割
- U-Net++：嵌套跳跃连接
- Attention U-Net：注意力机制
- V-Net：3D全卷积网络

**nnU-Net**
- 自适应配置
- 自动预处理
- 集成策略
- 多个数据集最佳性能

#### Transformer在分割中的应用

**UNETR**
- Transformer编码器 + CNN解码器
- 捕获全局依赖
- 3D医学图像分割

**Swin UNETR**
- Swin Transformer
- 层次化特征
- 移动窗口注意力

#### SAM（Segment Anything Model）

**特点**
- 零样本分割能力
- 提示驱动分割
- 通用分割模型

**医学影像应用**

```python
data = img.get_fdata()
affine = img.affine

# 保存NIfTI文件
new_img = nib.Nifti1Image(data, affine)
nib.save(new_img, 'output.nii.gz')
```

### 7.2 深度学习框架

#### MONAI（Medical Open Network for AI）

**特点**
- 基于PyTorch
- 专为医学影像设计
- PyTorch生态系统成员

**核心功能**
```python
import monai
from monai.transforms import Compose, LoadImage, NormalizeIntensity
from monai.networks.nets import UNet
from monai.losses import DiceLoss
from monai.metrics import DiceMetric

# 数据加载和预处理
transforms = Compose([
    LoadImage(image_only=True),
    NormalizeIntensity(),
])

# 定义网络
model = UNet(
    spatial_dims=3,
    in_channels=1,
    out_channels=2,
    channels=(16, 32, 64, 128, 256),
    strides=(2, 2, 2, 2),
)

# 定义损失函数
loss_function = DiceLoss(to_onehot_y=True, softmax=True)

# 定义评估指标
dice_metric = DiceMetric(include_background=False, reduction="mean")
```

**优势**
- 医学影像专用预处理

## 8. 前沿技术与趋势

### 8.1 多模态融合

#### 融合层次
1. **像素级融合**
   - 早期融合
   - 图像配准后融合

2. **特征级融合**
   - 中期融合
   - 特征提取后融合

3. **决策级融合**
   - 晚期融合
   - 模型输出融合

#### 应用
- PET-CT融合
- MRI多序列融合
- 影像-临床数据融合

### 8.2 联邦学习（Federated Learning）

#### 原理
```
本地数据 → 本地训练 → 上传梯度 → 聚合更新 → 下发模型 → 本地更新
```

#### 优势
- 数据隐私保护
- 数据不出医院
- 多中心协作

#### 挑战
- 数据异质性
- 通信效率
- 模型聚合策略

### 8.3 可解释AI（Explainable AI）

#### 方法
1. **注意力图（Attention Map）**
   - 显示模型关注区域
   - Grad-CAM

2. **显著性图（Saliency Map）**
   - 像素重要性
   - 输入梯度

3. **特征可视化**
   - 网络学到的特征
   - 激活最大化

#### 医学影像中的重要性
- 医生信任
- 临床采纳
- 错误诊断

### 8.4 生成模型

#### GAN（生成对抗网络）
- 医学图像生成
- 数据增强
- 图像翻译

#### Diffusion Model（扩散模型）
- 高质量图像生成
- 医学图像合成
- 图像修复

#### 应用
- 数据增强
- 隐私保护数据共享
- 缺失模态生成

### 8.5 自监督学习

#### 方法
- 对比学习
- 掩码图像建模
- 旋转预测

#### 优势
- 无需大量标注
- 学习通用特征
- 迁移学习
