## 1. 影像组学概述

### 1.1 定义与概念

**影像组学（Radiomics）** 是一种从医学影像（CT、MRI、PET等）中提取大量定量特征的方法，这些特征能够反映肿瘤的异质性，为临床决策提供客观、定量的依据。

**核心思想**：
- 医学影像包含肉眼无法识别的丰富信息
- 通过高通量计算提取定量特征
- 将影像转化为可挖掘的数据
- 辅助临床诊断、治疗和预后预测

### 1.2 发展历程

| 阶段 | 时间 | 特点 |
|------|------|------|
| 萌芽期 | 2010年前 | 纹理分析、形状特征 |
| 形成期 | 2012-2015 | 影像组学概念提出、特征标准化 |
| 发展期 | 2016-2020 | 深度学习融合、多中心研究 |
| 成熟期 | 2021至今 | 临床转化、多模态融合、AI集成 |

### 1.3 与传统影像诊断的区别

| 维度 | 传统影像诊断 | 影像组学 |
|------|-------------|---------|
| 分析方式 | 定性、主观 | 定量、客观 |
| 信息利用 | 可见特征 | 可见+不可见特征 |
| 可重复性 | 较低 | 较高 |
| 数据量 | 少量特征 | 高通量特征（数百到数千） |
| 临床应用 | 经验驱动 | 数据驱动 |

### 2.2 各步骤详解

#### 步骤1：图像获取

- **重采样**：统一体素大小（如1×1×1 mm³）
- **灰度离散化**：

  - 固定bin数量（如32或64）

  - Z-score归一化
  - Min-Max归一化

  - 对比度调整

- **分割方法**：

  - 3D Slicer
  - ITK-SNAP

#### 步骤4：特征提取

**特征分类**：

**A. 一阶统计特征（First-order Statistics）**
- 基于直方图的特征
- 描述灰度分布
- 包括：
  - 均值（Mean）
  - 方差（Variance）
  - 偏度（Skewness）
  - 峰度（Kurtosis）
  - 中位数（Median）
  - 百分位数（Percentiles）

**B. 形状特征（Shape-based Features）**
- 描述ROI的几何特性
- 包括：
  - 体积（Volume）
  - 表面积（Surface Area）
  - 球形度（Sphericity）
  - 紧凑度（Compactness）
  - 最大3D直径
  - 轴比

**C. 纹理特征（Texture Features）**

**灰度共生矩阵（GLCM - Gray Level Co-occurrence Matrix）**
- 描述灰度空间关系
- 24个特征，包括：
  - 能量（Energy）
  - 对比度（Contrast）
  - 相关性（Correlation）
  - 同质性（Homogeneity）
  - 熵（Entropy）
  - 差熵（Difference Entropy）

**灰度游程矩阵（GLRLM - Gray Level Run Length Matrix）**
- 描述相同灰度连续出现
- 16个特征，包括：
  - 短游程强调（Short Run Emphasis）
  - 长游程强调（Long Run Emphasis）
  - 灰度非均匀性（Gray Level Non-uniformity）

**灰度区域大小矩阵（GLSZM - Gray Level Size Zone Matrix）**
- 描述相同灰度连通区域
- 16个特征

**灰度依赖矩阵（GLDM - Gray Level Dependence Matrix）**
- 描述灰度依赖关系
- 14个特征

**邻域灰度差矩阵（NGTDM - Neighboring Gray Tone Difference Matrix）**
- 描述邻域灰度差异
- 5个特征

**D. 高阶特征**
- 小波变换特征
- LoG滤波特征
- 高斯滤波特征

#### 步骤5：特征筛选

**目的**：
- 去除冗余特征
- 选择最有预测价值的特征
- 防止过拟合
- 提高模型泛化能力

**方法**：

**A. 统计筛选**
- t检验 / Mann-Whitney U检验

- 相关性分析

**B. 机器学习方法**

- 主成分分析（PCA）

**C. 稳定性筛选**
- ICC（组内相关系数）
- 重复性分析
- 批次效应校正

#### 步骤6：模型构建

**传统机器学习模型**：

#### 纹理特征详解

**GLCM特征计算**：
```
GLCM矩阵：P(i,j,d,θ)
- i, j：灰度值
- d：距离（通常为1）
- θ：角度（0°, 45°, 90°, 135°）

常用特征：
1. 能量（Energy）= Σ P(i,j)²
2. 对比度（Contrast）= Σ (i-j)² P(i,j)
3. 相关性（Correlation）= Σ (i-μx)(j-μy)P(i,j) / (σxσy)
4. 熵（Entropy）= -Σ P(i,j) log P(i,j)
```

#### 决策曲线分析（DCA）

```python
def decision_curve_analysis(y_true, y_prob, thresholds=np.arange(0, 1, 0.01)):
    """
    决策曲线分析
    """
    net_benefits = []
    
    for threshold in thresholds:
        # 预测为阳性的样本
        predicted_positive = y_prob >= threshold
        
        # 真阳性
        tp = np.sum((predicted_positive == 1) & (y_true == 1))
        # 假阳性
        fp = np.sum((predicted_positive == 1) & (y_true == 0))
        
        # 计算净获益
        n = len(y_true)
        net_benefit = (tp / n) - (fp / n) * (threshold / (1 - threshold))
        net_benefits.append(net_benefit)
    
    return thresholds, net_benefits

# 计算DCA
thresholds, net_benefits = decision_curve_analysis(y_test, y_prob)

# 绘制
plt.figure(figsize=(8, 6))
plt.plot(thresholds, net_benefits, label='Model')
plt.axhline(y=0, color='k', linestyle='--', label='Treat All')
plt.axhline(y=0, color='r', linestyle='--', label='Treat None')
plt.xlabel('Threshold Probability')
plt.ylabel('Net Benefit')
plt.title('Decision Curve Analysis')
plt.legend()
plt.show()
```

### 3.5 列线图（Nomogram）构建

```python
import numpy as np
import matplotlib.pyplot as plt

def draw_nomogram(features, coefficients, feature_ranges):
    """
    绘制列线图
    
    参数:
    - features: 特征名称列表
    - coefficients: 特征系数
    - feature_ranges: 各特征的范围
    """
    fig, ax = plt.subplots(figsize=(12, 8))
    
    # 计算总分范围
    max_coef = max(abs(c) for c in coefficients)
    total_points_range = 100  # 标准化到0-100分
    
    # 绘制每个特征的刻度
    y_positions = np.arange(len(features))
    
    for i, (feat, coef, (min_val, max_val)) in enumerate(zip(features, coefficients, feature_ranges)):
        # 特征名称
        ax.text(-0.1, len(features) - i - 1, feat, ha='right', va='center', fontsize=10)
        
        # 特征刻度
        points = np.linspace(min_val, max_val, 5)
        scaled_points = (points - min_val) / (max_val - min_val) * abs(coef) / max_coef * total_points_range
        
        for j, (val, point) in enumerate(zip(points, scaled_points)):
            ax.text(point / total_points_range, len(features) - i - 1, f'{val:.2f}', 
                   ha='center', va='center', fontsize=8)
    
    # 总分刻度
    ax.text(1.1, len(features) / 2, 'Total Points', ha='left', va='center', fontsize=10, rotation=90)
    
    ax.set_xlim(-0.2, 1.3)
    ax.set_ylim(-0.5, len(features) + 0.5)
    ax.axis('off')
    
    plt.title('Nomogram', fontsize=14, fontweight='bold')
    plt.tight_layout()
    plt.show()

# 示例使用
features = ['Age', 'Tumor Size', 'GLCM_Entropy', 'Shape_Sphericity']
coefficients = [0.5, -0.3, 0.8, -0.6]
feature_ranges = [(30, 80), (1, 10), (0.5, 3), (0.6, 1.0)]

draw_nomogram(features, coefficients, feature_ranges)
```

## 4. 深度学习影像组学

### 4.1 深度学习特征提取

#### 预训练模型提取特征

```python
import torch
import torch.nn as nn
from torchvision import models, transforms
from PIL import Image

# 加载预训练模型
model = models.resnet50(pretrained=True)

# 移除最后的全连接层
model = nn.Sequential(*list(model.children())[:-1])
model.eval()

# 图像预处理
preprocess = transforms.Compose([
    transforms.Resize(256),
    transforms.CenterCrop(224),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], 
                        std=[0.229, 0.224, 0.225]),
])

# 提取特征
def extract_deep_features(image_path):
    image = Image.open(image_path).convert('RGB')
    image_tensor = preprocess(image).unsqueeze(0)
    
    with torch.no_grad():
        features = model(image_tensor)
    
    return features.squeeze().numpy()

# 使用
features = extract_deep_features('ct_slice.png')
print(f"Deep features shape: {features.shape}")
```

#### 3D CNN特征提取

```python
import torch
import torch.nn as nn

class RadiomicsCNN3D(nn.Module):
    def __init__(self, in_channels=1):
        super(RadiomicsCNN3D, self).__init__()
        
        self.features = nn.Sequential(
            nn.Conv3d(in_channels, 32, kernel_size=3, padding=1),
            nn.BatchNorm3d(32),
            nn.ReLU(inplace=True),
            nn.MaxPool3d(kernel_size=2, stride=2),
            
            nn.Conv3d(32, 64, kernel_size=3, padding=1),
            nn.BatchNorm3d(64),
            nn.ReLU(inplace=True),
            nn.MaxPool3d(kernel_size=2, stride=2),
            
            nn.Conv3d(64, 128, kernel_size=3, padding=1),
            nn.BatchNorm3d(128),
            nn.ReLU(inplace=True),
            nn.MaxPool3d(kernel_size=2, stride=2),
            
            nn.Conv3d(128, 256, kernel_size=3, padding=1),
            nn.BatchNorm3d(256),
            nn.ReLU(inplace=True),
            nn.AdaptiveAvgPool3d(1),
        )
```

### 4.3 端到端深度学习模型

```python
import torch
import torch.nn as nn

class RadiomicsNet(nn.Module):
    """
    端到端影像组学网络
    """
    def __init__(self, in_channels=1, num_classes=2):
        super(RadiomicsNet, self).__init__()
        
        # 特征提取
        self.features = nn.Sequential(
            nn.Conv3d(in_channels, 32, kernel_size=3, padding=1),
            nn.BatchNorm3d(32),
            nn.ReLU(inplace=True),
            nn.MaxPool3d(2),
            
            nn.Conv3d(32, 64, kernel_size=3, padding=1),
            nn.BatchNorm3d(64),
            nn.ReLU(inplace=True),
            nn.MaxPool3d(2),
            
            nn.Conv3d(64, 128, kernel_size=3, padding=1),
            nn.BatchNorm3d(128),
            nn.ReLU(inplace=True),
            nn.AdaptiveAvgPool3d(1),
        )
        
        # 分类器
        self.classifier = nn.Sequential(
            nn.Linear(128, 64),
            nn.ReLU(inplace=True),
            nn.Dropout(0.5),
            nn.Linear(64, num_classes),
        )
    
    def forward(self, x):
        x = self.features(x)
        x = x.view(x.size(0), -1)
        x = self.classifier(x)
        return x

# 训练
model = RadiomicsNet()
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)

# 训练循环
for epoch in range(100):
    model.train()
    optimizer.zero_grad()
    
    outputs = model(inputs)
    loss = criterion(outputs, labels)
    
    loss.backward()
    optimizer.step()
    
    if (epoch + 1) % 10 == 0:
        print(f'Epoch [{epoch+1}/100], Loss: {loss.item():.4f}')
```
